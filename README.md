## RecipeAppPoe
RecipeAppPoe is a simple console-based application written in C# that allows users to input recipes, duplicate them with different quantities, and revert to the original recipe if needed.

## Features
Recipe Entry: Users can input the ingredients, quantities, units of measurement, and descriptions for their recipes.
Recipe Duplication: Users can duplicate recipes with quantities multiplied by 0.5, 2, or 3.
Revert to Original Recipe: Users can revert to the original recipe after duplicating it.
User-Friendly Interface: The application provides clear prompts and instructions for user interaction.
